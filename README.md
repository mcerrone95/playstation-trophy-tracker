# PlayStation Trophy Tracker

A personal app to track PlayStation trophy progress, view completion percentages, and set gaming goals.

## Planned Features
- Trophy list with completion percentages
- Rarity stats per trophy
- Goal setting for target platinums
- Public-facing profile page

## Tech Stack
- Frontend: React
- Backend: Python
- Hosting: AWS S3 + CloudFront
- Database: DynamoDB (future)
- Data Source: PSNProfiles / Exophase
